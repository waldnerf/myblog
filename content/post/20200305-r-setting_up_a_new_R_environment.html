The wonders of the pacman package

The pacman package is a wonderful package that greatly facilitates the installation of all the packages that you need to run your project. Simply create a file (e.g., install_packages.R) with the following:

if (!require("pacman")) install.packages("pacman")
my_packages <- c('tidyverse', 'star', 'sf', 'caret')
pacman::p_load(my_packages)

And voila!
